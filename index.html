  • What Information is presented in App

    This app provides you with free alerts for Technical indicators like RSI. You can set alerts in app and when those level reaches, the app alerts you.
    Things to consider :
    1. The RSI values are 14 period daily values.
    2. More indicators will be added soon.
    3. You can set various notifications for each stock from Stock Details Screen. By default, you will be notified if the 14 period daily RSI value falls below 30 or rises above 70.
    4. The stock list is fixed for now but we intent to add more stock soon.
    4. The values in app are delayed and may not reflect the current level but we try our best to provide most latest data we have.
    5. For any queries, suggestions, please reach out to niara.android@gmail.com
